# Momon.Bijus - Jewelry E-commerce 💎

A professional e-commerce landing page and catalog developed for a real jewelry brand based in Brazil (Juazeiro-BA/Petrolina-PE). This project focuses on providing a high-end shopping experience with a clean, sophisticated design.

### 🚀 Key Features
- **Dynamic Catalog:** Integrated Isotope filtering for product categories (Rings, Earrings, Necklaces, Sets).
- **Responsive Interface:** Built with Bootstrap 5 to ensure a seamless experience across mobile and desktop.
- **Engagement Tools:** Testimonial sections, brand mission/vision, and direct WhatsApp integration for sales.
- **Visuals:** Implemented with AOS (Animate On Scroll) and GLightbox for a premium feel.

### 🛠️ Technical Stack
- **Frontend:** HTML5, CSS3 (Custom Main CSS), JavaScript.
- **Frameworks:** Bootstrap v5.3.3.
- **Libraries:** Isotope (filtering), Swiper (carousels), AOS (animations).
- **Collaboration:** Developed in partnership with a Back-end developer for future system integration.

### 📈 Future Data Analysis Opportunities
This project provides a perfect foundation for:
- **Sales Analytics:** Tracking which categories (Anéis vs Colares) have more engagement.
- **Conversion Rate:** Measuring button clicks (CTAs) for WhatsApp leads.
- **Customer Feedback Analysis:** Quantitative study of customer testimonials.
